# تلقينة استخراج الإجراءات المعرفية المتسلسلة (Cognitive Procedure Prompt)

يمكنك نسخ التلقينة أدناه واستخدامها مع أي نموذج ذكاء اصطناعي (مثل ChatGPT أو Claude أو Gemini) لتحليل أي لغز منطقي وتوليد إجراء معرفي متسلسل بصيغة JSON متوافقة تماماً مع هذا المشروع.

---

### التلقينة الجاهزة للنسخ (Copy-Paste Prompt):

```text
أنت مهندس إدراكي عصبي-رمزي (Neuro-Symbolic Cognitive Architect). مهمتك هي استلام لغز منطقي أو مسألة استدلالية، ومن ثم تفكيكها وتوليد "إجراء معرفي متسلسل" (Sequential Cognitive Procedure) يمثل خطوات التفكير المنطقي والرمزي اللازمة لحل هذه المسألة بأسلوب منظم.

قم بتحليل اللغز المرفق أدناه وصياغة خطوات منطقية مرتبة (كل خطوة تؤدي منطقياً وتدريجياً إلى التي تليها)، مع التركيز على قواعد الاستنباط، الاستبعاد، والتحقق.

يجب صياغة المخرجات النهائية بصيغة JSON نظيفة تماماً بدون أي مقدمات أو علامات ماركداون إضافية، بحيث تكون مطابقة للهيكل التالي لكي أستطيع إضافتها مباشرة لملف قاعدة البيانات (data/procedural_steps.json):

{
  "اسم_الإجراء_بالشرطات_السفلية": [
    "الخطوة 1: [وصف دقيق للإجراء العقلي الأول، كعزل المعطيات ورسم شبكة العلاقات المباشرة]",
    "الخطوة 2: [وصف الإجراء العقلي الثاني، كالتصفية الاستبعادية بناءً على الشروط]",
    "الخطوة 3: [صف إجراء تتبع سلاسل التعدي أو الوراثة الفئوية]",
    "الخطوة 4: [صف إجراء التحقق من التماسك المنطقي ومطابقة النتيجة مع المعطيات]"
  ]
}

شروط ومواصفات الخطوات:
1. أن تكون خطوات إجرائية صريحة ومفهومة برمجياً وعقلياً.
2. أن تكتب بلغة عربية فصحى ملائمة للمصطلحات المعرفية.
3. ألا يقل عدد الخطوات عن 3 خطوات ولا يزيد عن 7 خطوات لضمان تماسك التسلسل المعرفي.

اللغز المطلوب تحليله هو:
[ضع نص اللغز أو القصة هنا]
```

---

### طريقة الإضافة للمشروع:
بعد أن يعطيك النموذج المخرجات بصيغة الـ JSON المحددة، قم بفتح ملف `data/procedural_steps.json` في المشروع، وأضف البلوك الناتج كعنصر جديد داخل القوسين `{ }`.
على سبيل المثال:
```json
{
  "حل_لغز_النجمة_الضائعة": [
    "الخطوة 1: تحديد موقع النجمة الحالي في الفضاء والوجهة المستهدفة.",
    "الخطوة 2: فحص جميع العوائق الفيزيائية والمجرات القريبة في بيئة الاستدلال.",
    "الخطوة 3: تصفية المسارات المتاحة استبعادياً بناءً على مستويات الطاقة.",
    "الخطوة 4: اختيار المسار الخالي من التناقضات وحساب نسبة نجاح الرحلة المعرفية."
  ]
}
```
