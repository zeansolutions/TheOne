## ⚙️ تفاصيل واجهة البرمجة (Python API)

### كلاس `LanguageSelectionEngine`
الموجود في `src/manager/language_selection_engine.py`.

#### `__init__(self, languages_data)`
* **الوصف:** يهيئ اللغات المدعومة ويحدد اللغة الافتراضية للنظام كـ `"en"` (الإنجليزية).
* **المعاملات:**
  * `languages_data` (*dict* أو *str*): قاموس التكوين أو مسار ملف `languages.json`.

#### `detect_language(self, text)`
* **الوصف:** يفحص النصوص والرموز ويعيد رمز اللغة المكتشفة (مثل `"ar"` أو `"en"` أو `"fr"`).
* **المعاملات:**
  * `text` (*str*): نص السؤال المدخل.
* **المخرجات:** رمز اللغة المكتشفة (*str*).

#### `select_language(self, detected_lang, user_preference=None, conversation_history=None)`
* **الوصف:** يطبق ترتيب أولويات اختيار اللغة: `تفضيل المستخدم` $\to$ `اللغة المكتشفة` $\to$ `آخر لغة مستخدمة في السياق` $\to$ `اللغة الافتراضية`.
* **المخرجات:** رمز اللغة المعتمدة نهائياً (*str*).

## 🖥️ طريقة الاستخدام من التيرمينال
1. ابدأ واجهة التشغيل التفاعلية:
   ```bash
   ./start.sh run
   ```
2. يمكنك تغيير لغة واجهة التيرمينال بالكامل إلى أي لغة من اللغات الـ 11 المدعومة عبر اختيار الخيار رقم **5**، ثم كتابة رمز اللغة (مثال: اكتب `ar` للعربية، `en` للإنجليزية، `tr` للتركية، `zh` للصينية).
3. تترجم خيارات وبنرات لوحة التحكم تلقائياً للغة التي اخترتها مع الحفاظ على قدرة المحرك على إجابتك بنفس لغة سؤالك.
