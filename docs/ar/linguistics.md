# التحليل الصرفي والمطابقة اللغوية (Morphological Analysis)

## 📋 نظرة عامة
لمعالجة مدخلات اللغة الطبيعية محلياً دون الحاجة لنماذج الذكاء الاصطناعي التوليدي عند الاستعلام، يتضمن محرك **TheOne** **محلل صرفي لغوي**. بالنسبة للغة العربية، تم تطبيق **خوارزمية التجريد التكراري (Iterative Stripping Algorithm)** التي تقوم بتجريد السوابق (Prefixes مثل "ال"، "و"، "في") واللواحق (Suffixes مثل "هم"، "ين"، "ات") وتطبيق خرائط الجذور الصرفية لربط المفردات بمعرفات الكيانات الصحيحة في الرسم البياني.

## ⚙️ تفاصيل واجهة البرمجة (Python API)

### كلاس `GraphHandler` (البحث الصرفي)
دوال في كلاس `GraphHandler` للبحث والمطابقة اللغوية.

#### `dynamic_morphological_lookup(self, word, language="ar")`
* **الوصف:** ينظف الكلمة المدخلة، ويجرد منها الزوائد الصرفية تكرارياً، ويطابق النتيجة أو الجذر ببطاقات التعريف المتاحة للكيانات باللغة المستهدفة.
* **المعاملات:**
  * `word` (*str*): الكلمة المراد تحليلها وصرفها.
  * `language` (*str*): لغة الكلمة، والافتراضي `"ar"`.
* **المخرجات:** معرف الكيان النصي (*str*) (مثل `"feline_carnivore"` للكلمة المعرّفة `"الأسد"`) أو يعيد `None` في حال عدم وجوده.

#### بنية ملف قواعد الصرف (JSON)
يحدد السوابق واللواحق والجذور الصرفية في ملف `language_rules.json` كالتالي:
```json
"morphology": {
  "prefixes": ["ال", "و", "في", "بـ", "لـ", "كـ"],
  "suffixes": ["ه", "ها", "هم", "ين", "ات", "ون"],
  "roots": [
    {
      "id": "عوش",
      "patterns": ["يعيش", "تعيش", "يعيشون", "عيش"]
    }
  ]
}
```

## 🖥️ طريقة الاستخدام

### معالجة المدخلات:
عند طرح سؤال على النظام يحتوي على كلمات متصلة بزوايد لغوية، مثل:
`"هل الأسد يعيش في السافانا؟"`
يقوم المحلل الصرفي بالتفكيك:
* `"الأسد"` $\to$ تجريد سابقة `"ال"` $\to$ `"أسد"` $\to$ مطابقة للكيان `"feline_carnivore"`.
* `"يعيش"` $\to$ مطابقة للجذر `"عوش"` $\to$ مطابقة للعلاقة `"lives_in"`.
* `"السافانا"` $\to$ تجريد سابقة `"ال"` $\to$ `"سافانا"` $\to$ مطابقة للكيان `"savanna"`.

يترجم المحرك الجملة إلى علاقة ثلاثية منطقية: `(feline_carnivore, lives_in, savanna)`.

### التكامل مع الواجهة الرسومية (GUI):
في واجهة Electron الرسومية، يتم تشغيل المحلل الصرفي تلقائياً في الخلفية عبر خادم API عند كتابة الأسئلة أو تعليم العقل حقائق جديدة لتسهيل المطابقة الديناميكية للكيانات بأي من اللغات المدعومة.
