# المحلل الصرفي والبحث اللغوي (Morphological & Linguistic Analysis)

## 📋 نظرة عامة
لمعالجة وفهم مدخلات اللغة الطبيعية محلياً دون الحاجة لنموذج ذكاء اصطناعي إحصائي عند التشغيل، يحتوي نظام **TheOne** على **محلل صرفي للغة العربية**.
يقوم هذا المحرك بـ **التجريد التكراري (Iterative Stripping)** للكلمات من خلال حذف السوابق (مثل حروف العطف والجر والتعريف: "ال"، "و"، "في") واللواحق (مثل ضمائر الملكية والجمع: "هم"، "ين"، "ات")، ثم مطابقة الكلمة المجردة أو جذرها الصرفي مع الكيانات المسجلة بالرسم البياني.

## ⚙️ تفاصيل واجهة البرمجة (Python API)

### كلاس `GraphHandler` (البحث الصرفي)
دوال في `GraphHandler` للتحليل اللغوي الصرفي.

#### `dynamic_morphological_lookup(self, word, language="ar")`
* **الوصف:** يحلل الكلمة المدخلة، ويجردها تكرارياً من اللواصق، ثم يطابقها مع وسوم الكيانات (Labels) أو مع الجذور الصرفية والأنماط المعرفة.
* **المعاملات:**
  * `word` (*str*): الكلمة المراد تحليلها وصرفها.
  * `language` (*str*): لغة الإدخال، القيمة الافتراضية `"ar"`.
* **المخرجات:** يعيد معرف الكيان في الرسم البياني (مثال: يعيد `"feline_carnivore"` عند إدخال كلمة `"وبالأسد"`)، أو يعيد `None` في حال عدم المطابقة.

#### هيكل قواعد الصرف والأنماط في ملف JSON:
يتم تكوين السوابق واللواحق والجذور كالتالي:
```json
"morphology": {
  "prefixes": ["ال", "و", "في", "بـ", "لـ", "كـ"],
  "suffixes": ["ه", "ها", "هم", "ين", "ات", "ون"],
  "roots": [
    {
      "id": "عوش",
      "patterns": ["يعيش", "تعيش", "يعيشون", "عيش"]
    }
  ]
}
```

## 🖥️ طريقة الاستخدام من التيرمينال
1. اسأل النظام أي سؤال باستخدام كلمات مصرفة أو مرتبطة بحروف عطف أو تعريف، مثل:
   `"هل الأسد يعيش في السافانا؟"`
   يقوم المحلل الصرفي بالتفكيك كالتالي:
   * `"الأسد"` $\to$ تجريد `"ال"` $\to$ `"أسد"` $\to$ يطابق الكيان `"feline_carnivore"`.
   * `"يعيش"` $\to$ يطابق الجذر الصرفي `"عوش"` $\to$ يطابق العلاقة `"lives_in"`.
   * `"السافانا"` $\to$ تجريد `"ال"` $\to$ `"سافانا"` $\to$ يطابق الكيان `"savanna"`.
2. يتم تحويل الجملة لعلاقة ثلاثية دلالية: `(feline_carnivore, lives_in, savanna)`.
